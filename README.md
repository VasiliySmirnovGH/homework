## 📌 Описание

Консольное приложение на **C#** (.NET 8), реализующее алгоритм поиска с опечатками на основе **расстояния Дамерау–Левенштейна** (алгоритм Вагнера–Фишера).

Программа:
- вычисляет минимальное количество операций (вставка, удаление, замена, транспозиция) для преобразования одной строки в другую;
- работает в бесконечном цикле ввода двух строк;
- выходит из цикла при вводе `exit` в качестве первой строки.

## ⚙️ Алгоритм

Алгоритм Вагнера–Фишера строит матрицу D[i,j] — редакционное расстояние между первыми `i` символами строки S1 и первыми `j` символами строки S2.

Поддерживаемые операции (каждая стоит 1):

| Обозначение | Операция |
|:-----------:|----------|
| `I` | Insert — вставка символа |
| `D` | Delete — удаление символа |
| `R` | Replace — замена символа |
| `T` | Transposition — перестановка двух соседних символов (поправка Дамерау) |

Временная сложность: **O(M × N)**  
Пространственная сложность: **O(M × N)**

## 📚 Контрольные вопросы

<details>
<summary><b>1. Что такое расстояние Левенштейна?</b></summary>

Расстояние Левенштейна (редакционное расстояние) — минимальное количество элементарных операций редактирования (вставка, удаление, замена символа), необходимых для преобразования одной строки в другую. Вычисляется методом динамического программирования через матрицу D[i,j].

</details>

<details>
<summary><b>2. Что такое расстояние Дамерау–Левенштейна?</b></summary>

Расширение расстояния Левенштейна, предложенное Фредериком Дамерау. Добавляет четвёртую операцию — **транспозицию** (перестановку двух соседних символов). Дамерау показал, что это наиболее частая ошибка при наборе текста. Транспозиция, которая стоила бы 2 в классическом расстоянии (две замены), стоит 1.

</details>

<details>
<summary><b>3. Алгоритм Вагнера–Фишера</b></summary>

Алгоритм заполняет матрицу D размером (M+1)×(N+1):
1. Инициализация: D[i,0] = i, D[0,j] = j
2. Заполнение: D[i,j] = min(вставка, удаление, замена)
3. Поправка Дамерау: если S1[i]=S2[j-1] и S1[i-1]=S2[j], то D[i,j] = min(D[i,j], D[i-2,j-2]+m)
4. Ответ: D[M,N]

</details>
